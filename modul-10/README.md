## ğŸ“– Deskripsi Singkat
Proyek ini dibuat sebagai bagian dari praktikum **Mobile Programming - Modul 10**, yang membahas penerapan:

- Desain antarmuka (GUI) menggunakan widget dasar dan layouting pada Flutter.
- Navigasi antar halaman menggunakan **Named Route** untuk menjaga struktur aplikasi lebih rapi.
- Pembuatan **Model Data** dan proses **JSON Serialization / Deserialization**.
- Integrasi data dari API eksternal menggunakan paket `http`.
- Penampilan daftar data dalam bentuk **ListView**, serta halaman detail dari item yang dipilih.

Aplikasi ini mengambil data **daftar game gratis** dari API eksternal dan menampilkannya pada halaman utama. Saat salah satu item dipilih, pengguna diarahkan ke halaman detail dengan informasi lengkap mengenai game tersebut.
